# COMP2001 API

These are the files for the MVC layer of the API. It includes

* A Programmes Controller - where it has the HTTP Methods; GET, POST, PUT AND DELETE.
* A DataAccess File which executes the 3 Stored Procedures to help the API for POST, PUT and DELETE.
* A Programmes Model File which gets the content of the programmes table.
